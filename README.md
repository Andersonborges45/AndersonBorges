# Sistema de Gestão de Estoque de Veículos

Projeto desenvolvido utilizando:

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Funcionalidades

- Cadastro de Veículos
- Cadastro de Marcas
- Cadastro de Modelos
- Consulta de Veículos
- Filtros por Marca
- Filtros por Modelo
- Filtros por Ano
- Filtros por Status
- Atualização de Dados
- Remoção de Veículos
